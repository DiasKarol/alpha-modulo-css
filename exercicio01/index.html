O que é e para que serve a CSS

CSS significa Cascading Stylesheets(Folhas de Estilo em Cascata), é uma linguagem criada para fazer estilização de páginas web. Com ela podemos dar cor aos elementos presentes na pagina, organizar, distribuir e arquitetar uma pagina web.

A sintaxe utilizada na CSS é basicamente o seletor (elemento html como tags, classe, id, etc.) e logo após colocamos chaves e dentro teremos, uma propriedade (a regra que será aplicada como color, background, font-size, etc.) e iremos separar por dois pontos e informamos o valor (cor, tamanho da fonte, etc.), antes de fechar a nossa chave, colocamos ponto e virgula. Como podemos ver no exemplo abaixo.

p {
font-size: 24px;
color: green;
}


MANEIRAS DE INSERÇÃO

Temos 3 maneiras básicas de inserir a css em nosso documento html: inline, internal e external. iremos ver a seguir sua prioridade e como inserir.


CSS inline

No css inline, você atribui estilização em um elemento html específico.

O recomendado é que seja utilizado somente em casos que você não tenha acesso ao arquivo css, ou nos momentos em que for necessario inserir estilo a um unico elemento.

<p style="color: blue;"> Este texto seria apresentado na cor indicada no style </p>


CSS internal

No css internal, realizamos a atribuição de estilo a uma única página html.

Uma vantagem da utilização do internal, é que não vemos mais de um arquivo para a mesma páginas, o que também é uma desvantagem, por acabar levando mais tempo para carregar a página, ja que o documento ficará maior

<head>...</head>
<style>
body {
background-color: ocean;
color: blue;
}
</style>
<body>...</body>


CSS external

No css external, você consegue fazer o link de página html com o arquivo css

A vantagem do .css é que o arquivo estará em seu próprio documento, o que facilita na estilização de um grande site, já que podemos realizar a linkagem de um mesmo arquivo em outras paginas, economizamos tempo e unificamos o estilo utilizado.

Como utilizar:
Crie um documento com a extensão .css e faça toda a estilização do site. No documento html, voce irá linkar o documento .css no head de sua página.
Dessa forma:

<head>
<link rel="stylesheet" href="namedoc.css">
...
</head>
<body>...</body>

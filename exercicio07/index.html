Cores

As cores em CSS podem ser especificadas pelos seguintes métodos:

  • Cores hexadecimais
  • Cores hexadecimais com transparência
  • Cores RGB
  • Cores RGBA
  • Cores HSL
  • Cores HSLA
  • Cores predefinidos/entre navegadores
  • Com a palavra- currentcolorchave

Cores hexadecimais

Uma cor hexadecimal é especificada com: #RRGGBB, onde os inteiros hexadecimais e todos os valores devem estar entre 00 e FF.
Para adicionar transparência ao hexadecimal, basta acrescentar dois dígitos adicionais entre 00 e FF ao final do código.

  • RR (vermelho) #ff0000
  • GG (verde) #00ff00
  • BB (azul) #0000ff
  • Azul com transparência #0000ff80

Exemplo:

Este texto esta com cor hexadecimal de código #FF1493 e a sua marcação contém o código #ffb6c18a

Cores RGB

Define cores usando o modelo Red-green-blue, o RGB pode representaro total de 16.777.216 cores.
Para adicionar transparência ao RGB, basta acrescentar o(s) digito(s) alpha ao final do código.

  • rgb(255,0,0) -> red
  • rgb(0,255,0) -> green
  • rgb(0,0,255) -> blue
  • rgba(255, 0, 0, 1) -> vermelho com opacidade 100%

Exemplo:

Este texto esta com cor RGC de código rgb(0,255,255) e a sua marcação contém o código RGBA rgba(0, 206, 209, 0.596)

Cores HSL

HSL significa matiz, saturação e leveza e representa uma representação de cores de coordenadas cilíndricas.
Para adicionar transparência ao HSL, basta acrescentar o(s) digito(s) alpha ao final do código.

  • hue -> Define um grau na roda de cor (de 0 a 360) - 0 (ou 360) é vermelho, 120 é verde, 240 é azul
  • saturation -> Define a saturação; 0% é um tom de cinza e 100% é a cor completa (saturação completa)
  • lightness -> Define a leveza; 0% é preto, 50% é normal, e 100% é branco

Exemplo:

Este texto esta com cor hsl de código HSL(120, 100%, 25%) e a sua marcação contém o código HSLA hsl(120, 60%, 70%, 0.5)

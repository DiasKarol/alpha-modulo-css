Estilizando links

:link

Seleciona os links dentro de um elemeto, Um link que tem um destino.

Por regra, o link tem que vir antes de: :visited — :hover — :active

:visited

Indica se o link já foi visitado pelo usuário (se já existe no histórico do navegador)

:hover

Estiliza o elemento quando selecionado

:focus

Quando um elemento recebe foco, ou seja, quando o usuário seleciona o elemento utilizando o teclado ou ativando com o mouse

:active

Um link que é ativado, clicado

Por regra, o link tem que vir depois de: :link — :visited — :hover

Pode ser utilizado com qualquer elemento

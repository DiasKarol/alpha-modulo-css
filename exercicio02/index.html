Seletores

O seletore em css é o primeiro elemento de um conjunto de regras utilizadas no documento. Servem para informar ao navegador qual elemento será estilizado.


Seletor universal

Ele seleciona todo o documento e são representados por um *. É muito utlizado para "resetar" a folha de estilo e retirar todo o estilo padrão ja definido pelo navegador.

Exemplo:

* {
margin: 0;
padding: 0;
}

Seletor tag/elemento

O que ira referenciar o seletor é o proprio elemento html, ou seja, um h1, h2, section podem ser utilizados no seletor. Geralmente usados para estilização de repetição.

Por exemplo, se eu quero que todos os meus h2 tenham a cor purple e não quero realizar esse procedimento em cada h2 isolados, farei essa seleção atraves do css internal ou external desta forma:

h2 {

color: black;

}

Seletor Class

Para utilizar uma class como seletor, primeiro precisamos entender que a class pode se repetir em mais de elemento e que os elementos podem ter mais de uma class

O seletor class é referenciado no css por um . (ponto) antes do nome atribuido a class.

Exemplo:

.myclass {

color: darkgreen;
background-color: lightgreen;

}

Seletor ID

O ID age como uma identidade, dessa forma, o id não pode ser repetido e só podemos inserir uma ID em cada elemento.

O seletor ID é referenciado no css por um # antes do nome atribuido a ele.

Exemplo:

#myid {

color: darkviolet;
background-color: violet;

}

Seletor de atributo

Seleciona os elementos através de um atributo dado a tag html

    Podemos selecionar elementos que o valor do atributo sejam:

  • Exatamente iguais ao valor informado: p[atributo="valor"]
  • Iniciados com o valor informado: p[atributo^="valor"]
  • Finalizem com o valor informado: p[atributo$="valor"]
  • Que contenha o valor informado: p[atributo*="valor"]

Pseudo-class

As pseudo-classes funcionam como palavras-chaves que são adicionadas ao seletor de elementos no qual se encontram em um estado específico.

Por exemplo, se eu quiser que o primeiro paragrafo de um artigo fique com a cor azul, posso adicionar um :first-child e ir atualizando somente o html a partir deste ponto.

no css ficaria assim:
article:first-child {
color: blue;
}

Pseudo-elementos

As pseudo-elementos são como palavras-chaves adicionadas ao seletor para estilizar uma parte especifica do elemento selecionado.

Por exemplo: para adicionar um simbolo antes de um link, usamos:
a::before {
content: "⇒";
}

combinator

Como o proprio nome diz, este seletor combina outros dois seletores dando uma melhor localização dentro do documento.

    Existem 4 combinators diferentes:

  • O combinador descendente (Espaço): é feito no relacionamento de pai, pai do pai, pai do pai do pai...
  • O Filho (>): Corresponde somente aos filhos diretos do primeiro seletor
  • O irmão adjacente (+): Corresponde apenas aos elementos correspondidos pelo segundo seletor que é o próximo elemento irmão do primeiro seletor
  • O irmão geral (~): Ele seleciona irmãos de um elemento, mesmo que eles não sejam diretamente adjacentes.

Caixas

Sabemos que em uma página html, todo elemento se comporta como caixa, hoje iremos estudar sobre esses comportamentos e como utiliza-los em css.

Box model

O modelo de caixa CSS define como as diferentes partes de uma caixa trabalham juntas para criar uma caixa que você pode ver em uma página.

As propriedades

Width
Determina a largura da área de conteúdo (content) de um elemento.

Height
Determina a altura da área do conteúdo de um elemento.

Padding
Define uma a distância entre o conteúdo de um elemento e suas bordas.

Margin
Define a área de margem nos quatro lados do elemento.
É a distância entre o elemento em si e os elementos vizinhos.

Alguns tipos de Display

O valor do display define o tipo de renderização que será usada no elemento.

Block
O elemento gera uma caixa de elemento de bloco.

Inline
O elemento gera uma ou mais caixas de elementos inline.

Flex
O elemento se comporta como um elemento de bloco e apresenta seu conteúdo de acordo com o modelo Flexbox. Como o nome sugere, é flexível, o que permite a fácil adaptação dos elementos no espaço disponível.

grid
O elemento se comporta como um elemento de bloco e apresenta seu conteúdo de acordo com o modelo de grade, ou seja é definido uma estrutura de linha e coluna.

Overflow

Overflow significa transbordar em português.

Como ja sabemos, tudo em css é contido em uma caixa, que podemos controlar seu tamanho atraves de propriedades fornecidas pelo css.

Quando uma caixa é menor que seu conteúdo, as informações transbordam e o css fornece a ferramenta overflow para solucionar este problema.

Para isso temos 4 tipos de overflow:

Visible

Evita a perda de dados.
